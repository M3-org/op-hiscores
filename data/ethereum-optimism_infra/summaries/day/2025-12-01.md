# ethereum-optimism/infra Daily Update (Dec 1, 2025)

## OVERVIEW 
The day saw significant progress in the `op-acceptor` tool with the addition of multi-gate input and dry-run functionalities, enhancing its testing and reporting capabilities. A new pull request was opened to streamline CI workflows.

## KEY TECHNICAL DEVELOPMENTS

### Enhanced `op-acceptor` Functionality
The `op-acceptor` tool received a significant upgrade with the introduction of multi-gate input and a dry-run feature. This allows for more flexible testing by enabling multiple gates to be passed as input and provides a dry-run option for pre-execution validation. ([#520](https://github.com/ethereum-optimism/infra/pull/520))

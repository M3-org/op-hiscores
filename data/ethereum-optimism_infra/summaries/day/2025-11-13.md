# ethereum-optimism/infra Daily Update (Nov 13, 2025)
## OVERVIEW 
Today's development focused on enhancing CI/CD with new GitHub Actions workflows for Docker image attestation and signing, alongside a minor version bump for `op-acceptor`.

## KEY TECHNICAL DEVELOPMENTS

### Enhanced CI/CD for Docker Images
- Added comprehensive GitHub Actions workflows to build, attest, and sign Docker images for all infrastructure components, significantly improving supply chain security and integrity. [#510](https://github.com/ethereum-optimism/infra/pull/510)

### Infrastructure Component Updates
- Released `op-acceptor` version 3.6.6 with a fresh commit to facilitate a new release tag. [#509](https://github.com/ethereum-optimism/infra/pull/509)

- **Kona Light CL Support**: Support for Kona light CL was added to `op-devstack`, allowing for more flexible configuration of L2 consensus clients. [ethereum-optimism/optimism#18627](https://github.com/ethereum-optimism/optimism/pull/18627)

### Deployment and Transaction Management Enhancements
- **OPCM v2 Support for `op-deployer`**: The `op-deployer` tool received a significant upgrade with the addition of OPCM v2 support for chain upgrades, enabling more robust and flexible deployment processes. [ethereum-optimism/optimism#18593](https://github.com/ethereum-optimism/optimism/pull/18593)
- **`op-deployer` Max Code Size Override**: `op-deployer` will now ignore max-code-size limits during deployment, streamlining the developer experience for devnet setups. [ethereum-optimism/optimism#18642](https://github.com/ethereum-optimism/optimism/pull/18642)
- **New `BlobGasPriceOracle` Module**: A new `BlobGasPriceOracle` module was introduced to `op-service`, designed to track and cache blob prices for more efficient transaction management. [ethereum-optimism/optimism#18386](https://github.com/ethereum-optimism/optimism/pull/18386)

- **Comprehensive Fuzz Testing**: Heavy fuzz tests now incorporate the dev feature matrix for broader coverage. ([ethereum-optimism/optimism#18368](https://github.com/ethereum-optimism/optimism/pull/18368))
- **CI Stability Fixes**: A fix was implemented to skip validation for missing test artifacts, improving CI stability. ([ethereum-optimism/optimism#18367](https://github.com/ethereum-optimism/optimism/pull/18367))
- **Resource Upgrade for E2E Tests**: The `op-e2e-cannon-tests` job was upgraded to use a larger machine to prevent resource-related failures. ([ethereum-optimism/optimism#18359](https://github.com/ethereum-optimism/optimism/pull/18359))

### Contract and Release Management
- **Safe Directory Consistency**: All Safe directory changes from the `develop` branch were cherry-picked to the `op-safe-contracts/v1.0.0` branch. ([ethereum-optimism/optimism#18371](https://github.com/ethereum-optimism/optimism/pull/18371))
- **Reliable Release Binary Provider**: The GitHub release binary provider was updated with version checks and a static cache pather for improved reliability. ([ethereum-optimism/optimism#18369](https://github.com/ethereum-optimism/optimism/pull/18369))

### CI/CD Workflow Enhancements and Fixes

([#522](https://github.com/ethereum-optimism/infra/pull/522), [#519](https://github.com/ethereum-optimism/infra/pull/519))
- **CI Workflow Fixes**: Updated `docker-bake.yaml` to resolve bash history expansion issues, ensuring more reliable CI processes in `optimism`. ([#18498](https://github.com/ethereum-optimism/optimism/pull/18498))

### Contract Optimization and Testing
- **OPContractsManagerV2 Refactoring**: Broke out `OPContractsManagerV2` utilities into a separate contract, reducing code size and improving modularity in `optimism`. ([#18454](https://github.com/ethereum-optimism/optimism/pull/18454))
- **Enhanced Contract Test Coverage**: Added dedicated test coverage for `AddressAliasHelper` and fixed a flaky test in `SystemConfig` in `optimism`. ([#18360](https://github.com/ethereum-optimism/optimism/pull/18360))

### Documentation and Configuration Updates
- **Documentation Formatting and Content Updates**: Fixed Nextra to Mintlify tab formatting, updated Actions SDK configuration, clarified batcher throttling descriptions, and added Jovian Network Upgrade information in `docs`. ([#1901](https://github.com/ethereum-optimism/docs/pull/1901), [#1869](https://github.com/ethereum-optimism/docs/pull/1869), [#1902](https://github.com/ethereum-optimism/docs/pull/1902), [#1900](https://github.com/ethereum-optimism/docs/pull/1900))
- **Prestate Configuration Updates**: Added Kona 1.2.5 prestates to `standard-prestates.toml` in `superchain-registry`. ([#1174](https://github.com/ethereum-optimism/superchain-registry/pull/1174))

### Data Integrity and Analytics
- **Duplicate Blob Hash Detection**: Updated blob fetching logic to correctly handle duplicate blobs with identical content but different indices in `optimism`. ([#18492](https://github.com/ethereum-optimism/optimism/pull/18492))
- **Collective Revenue Calculation Fix**: Fixed a bug affecting the collective revenue write process in `op-analytics` to ensure accurate financial data. ([#1772](https://github.com/ethereum-optimism/op-analytics/pull/1772))

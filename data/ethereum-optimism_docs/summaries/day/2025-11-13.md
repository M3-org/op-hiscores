# ethereum-optimism/docs Daily Update (Nov 13, 2025)
## OVERVIEW 
Today's focus was on enhancing documentation for chain operators, particularly around DA Footprint and Flashblocks RPC, alongside minor code formatting fixes.

([#1864](https://github.com/ethereum-optimism/docs/pull/1864), [#1858](https://github.com/ethereum-optimism/docs/pull/1858))

([#1863](https://github.com/ethereum-optimism/docs/pull/1863), [#1862](https://github.com/ethereum-optimism/docs/pull/1862))
